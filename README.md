# aloe
A chess engine.

## Installation
Use Rust's package manager, `cargo`.

Install or upgrade `aloe`:
```text
cargo install --git https://github.com/clfs/aloe
```

Uninstall `aloe`:
```text
cargo uninstall aloe
```
